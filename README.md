# FixLink

FixLink is a mobile-first marketplace for mechanics and spare parts sellers in Abuja, built with React, TypeScript, Vite, and Supabase-ready data flows.

## Features

- Mechanic product discovery and search
- Cart and checkout flow
- Spare part request workflow
- Seller dashboard and inventory views
- Supabase-aware auth and data persistence with a local fallback for demos

## Local setup

1. Install dependencies with `npm install`
2. Create a `.env` file in the project root with:
   - `VITE_SUPABASE_URL=your_supabase_project_url`
   - `VITE_SUPABASE_ANON_KEY=your_supabase_anon_key`
3. Run `npm run dev`

## Build

- `npm run build`

## Deployment

The app is ready to be deployed to Vercel or Netlify. Set the same environment variables in the host dashboard.
